# Installation

1. Repo를 Clone 해주세요.

```
git clone https://github.com/jayjunn/aitrics.git
cd repo (저장한 파일명)
```

2. 루트폴더에서 Dependencies를 설치해주세요

```
yarn install
```

3. 루트폴더에서 서비스를 실행해주세요
```
cd aitrics
yarn start
```
